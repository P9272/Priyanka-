This Khichadi_Device file is an Arduino File.
Arduino IDE 1.8.5 is used to write and execute this code.
To use this file we need to install the following libraries:
1. OneWire
2. LiquidCrystal_I2C
3. ADAFRUIT_FINGERPRINT

After installing these libraries, we have uploaded
 the code on the Arduino Uno Microcontroller kit.
